# 🀄 雀友計數機 HK Mahjong Calculator

一個專為**香港麻雀**設計嘅番數同輸贏計數 web app，色彩鮮明、操作簡單，最啱朋友聚會打牌時用！

**🎮 立即開玩：** <https://bensonlsp.github.io/hkmahjong-calculator/>

## ✨ 功能

### 💰 計錢

- **兩種計法任揀**
  - 傳統倍增：自訂起糊底價，支援「辣辣上」（每番×2）同「半辣上」（3番$8 → 4番$16 → 5番$24 → 6番$32 → 7番$48…）
  - 簡單模式：幾錢一番，直接乘
- **封頂（爆棚）**：8 番／10 番／13 番／無封頂
- **起糊番數**：3 番起／1 番起／無限制，未夠番會提你
- **出沖計法**：半銃（出沖畀一半，另外兩家各四分一）或全銃（出沖一個人孭晒）
- **自摸計法**：三家各畀半價或全價
- 撳掣前即時預覽糊牌價同每家畀幾多

### 🎯 打牌流程

- 顯示而家打緊咩風咩局（東風東局、南風西局…），一撳過莊
- 莊家有紅色「莊」標記，莊家食糊會提你連莊
- 揀邊個食糊 → 輸入番數（有 3/4/5/6/7/8/10/13 快捷掣）→ 揀自摸定出沖 → 一掣記低
- 四位玩家可改名，輸贏即時更新（綠色贏、紅色輸）
- 完整對局紀錄，撳錯可以「復原上一鋪」

### 📖 番數表

內置常見香港牌例參考表，由雞糊到十三幺：

|牌型        |番數      |
|----------|--------|
|平糊        |1 番     |
|對對糊／混一色   |3 番     |
|小三元       |5 番     |
|清一色       |7 番     |
|大三元       |8 番     |
|十三幺／大四喜／天糊|13 番（例牌）|


> ⚠️ 各家各法！部分牌型（坎坎糊、字一色、小四喜等）唔同枱有唔同打法，開枱前記得同雀友夾掂。
